#UniReserva - Sistema de Reservas de Recursos Universitarios
Producto Mínimo Viable (MVP) de una aplicación web para la gestión de reservas de salones, laboratorios y equipos universitarios.

#Descripción del Producto

UniReserva es una aplicación web de una sola página (SPA) que permite a estudiantes y docentes de la Universidad Mariano Gálvez de Guatemala reservar recursos académicos (salones, laboratorios y equipos).

#Características Principales:

- Consulta de disponibilidad en tiempo real- Visualiza qué recursos están libres en el horario deseado
- Reserva instantánea - Confirma tu reserva en segundos sin aprobaciones manuales
- Validación automática de conflictos - El sistema previene reservas duplicadas del mismo recurso
- Persistencia de datos - Tus reservas se guardan localmente y sobreviven al cierre del navegador
- Interfaz intuitiva y responsiva- Diseño adaptable a dispositivos móviles y escritorio
- Sin instalación requerida - Funciona directamente en el navegador web

#Tecnologías Utilizadas

- HTML5 Estructura semántica de la aplicación 
- CSS3 Estilos, layout responsivo (Flexbox/Grid), variables CSS 
- JavaScript Lógica de negocio, manipulación del DOM, eventos
- localStorage Web Storage API, Persistencia de datos entre sesiones
- Git Control de versiones 
- GitHub Pages Hosting y despliegue automático 

Arquitectura Técnica
- Frontend: Single Page Application (SPA) 100% client-side
- Persistencia:localStorage del navegador (JSON format)
- Despliegue: Servidor web estático (GitHub Pages)
